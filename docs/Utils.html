    module Utils

    Overview

    Defines general program utility methods. extend self allows these methods to be included in a program (class) and invoked without a namespace or just used as a namespace.

    def clear_screen #

    Clears the screen using ansi codes.

    \e[ is the escape code
2J clears the screen
1;1H moves cursor to line 1, column 1
    clear_screen # => "\e[2J\e[1;1H]]"
