    def resolve_host(host) #

    Does a lookup of a host's ip address and returns it as a Socket::IPAddress object. Raises an error if the hostname doesn't exist or can't be resolved. Remove the ":7" part with rstrip(":7") to get just the ip address.

    Helpers.resolve_host("example.com")                   # => ip_address:7
Helpers.resolve_host("example.com").to_s.rstrip(":7") # => ip_address
